## 📖 Overview

**Strawhats Engineering** is a premium, high-performance web platform built to showcase state-of-the-art software architecture, cloud migration capabilities, quantum cybersecurity grids, and AI core infrastructure.

Engineered with uncompromising visual excellence and sub-second load times, the project adopts a **Pure Standards Architecture**—built exclusively using **HTML5, Vanilla CSS3, and ES6 JavaScript Modules** without heavy monolithic bundling frameworks.

---

## 🚀 Key Features & Architectural Highlights

- **⚡ Zero-Build Architecture**: Uses native browser ES6 Modules and Vanilla CSS, eliminating Webpack/Vite compilation overhead during development.
- **🎨 Modern Premium Aesthetics**: Features sleek dark mode styling (`#05050A`), glassmorphism cards, glowing gradients, magnetic button wrappers, and an interactive 0-100% counter preloader.
- **🌀 Smooth Inertial Scrolling**: Integrated with **Lenis Scroll** for buttery-smooth scrolling across all devices.
- **✨ High-Performance Animations**: Powered by **GSAP (GreenSock)** and **ScrollTrigger** for hardware-accelerated DOM timelines and parallax section reveals.
- **♿ Accessibility & SEO Ready**: Semantic HTML5 sectioning, full WAI-ARIA compliance, keyboard navigation focus states, and OpenGraph metadata.

## 🛠️ Quick Start & Local Development

Because this project uses native web standards, you can run it locally with zero dependency installation or bundling required.

### 1. Using Python (Recommended)
```bash
# Navigate to the project directory
cd d:\Projects\strawhats

# Start the built-in HTTP server on port 8000
python -m http.server 8000
```
Open your browser and visit: **http://localhost:8000**

### 2. Using Node.js / npx
```bash
# Start a lightweight Node HTTP server
npx -y http-server -p 8000
```
